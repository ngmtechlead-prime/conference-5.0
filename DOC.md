# NGM 5.0 Conference Website Portal

## Project Overview

The NGM 5.0 Conference Website is a web portal for the Nasir Giwa Mentorship 5.0 Conference — a premier convergence of next-generation thinkers, builders, and creators. The website serves as the primary platform for conference information, ticket purchases, and competition registrations.

The conference features two major competitions:

1. **DARE Nigeria Challenge** (NGM X Dr. Kola Adesina Competition) — A pitch competition for Nigerian innovators aged 18-40 with bold ideas that can change Nigeria
2. **SME Pitch Competition** — A competition for small and medium enterprises

## Target Audience

- Aspiring entrepreneurs and innovators (18-40 years old)
- Nigerian citizens with startup ideas or prototypes
- Conference attendees and ticket buyers
- Potential sponsors and partners
- SME owners looking to pitch their businesses

---

## User Stories

### Conference Attendee

- As a **conference attendee**, I want to view the conference agenda so that I can plan which sessions to attend
- As a **conference attendee**, I want to see the list of speakers so that I know who will be presenting
- As a **conference attendee**, I want to buy tickets online so that I can attend the conference
- As a **conference attendee**, I want to browse the gallery so that I can see highlights from previous events
- As a **conference attendee**, I want to contact the organizers so that I can ask questions about the event

### Competition Applicant (DARE Nigeria Challenge)

- As an **applicant**, I want to learn about the DARE Nigeria Challenge so that I understand what the competition offers
- As an **applicant**, I want to see the eligibility criteria so that I know if I qualify to apply
- As an **applicant**, I want to view the prize amounts so that I understand what I could win
- As an **applicant**, I want to read FAQs so that I can get answers to common questions
- As an **applicant**, I want to register for the competition through a multi-step form so that I can submit my application
- As an **applicant**, I want to provide my personal information so that organizers can verify my eligibility
- As an **applicant**, I want to describe my innovation and pitch so that judges can evaluate my idea
- As an **applicant**, I want to upload supporting documents (pitch deck, video, ID) so that I can strengthen my application
- As an **applicant**, I want to indicate my availability and commitment so that organizers know I'm serious
- As an **applicant**, I want to digitally sign and submit my application so that I can complete my registration

### Competition Applicant (SME Pitch)

- As an **SME owner**, I want to learn about the SME Pitch Competition so that I can participate
- As an **SME owner**, I want to register my business for the pitch competition so that I can compete

### Sponsor

- As a **potential sponsor**, I want to learn about sponsorship opportunities so that I can support the conference
- As a **potential sponsor**, I want to contact the team so that I can discuss partnership options

### General Visitor

- As a **visitor**, I want to navigate easily between pages so that I can find information quickly
- As a **visitor**, I want to access the website on mobile so that I can browse on any device
- As a **visitor**, I want to view terms and privacy policies so that I understand my rights

## Competition Details

### DARE Nigeria Challenge

**Eligibility:**
- Nigerian citizens aged 18-40
- Innovation at idea or prototype stage
- No institutional funding received
- Must have conference ticket

**Prizes:**
- 1st Place: ₦2,000,000
- 2nd Place: ₦10,000,000
- 3rd Place: ₦1,000,000

**Application Steps:**
1. Applicant Profile (personal info, education, ID upload)
2. Innovation & Pitch (project details, SDGs, problem/solution)
3. Funding & Media (support needs, document uploads, video pitch)
4. Commitment (availability, declaration, digital signature)
